Overview
The Income Expense Calculator is a simple web app that helps users track their income and expenses. Users can add, edit, delete, and filter financial entries, and view their total income, total expenses, and net balance. Data is stored locally in the browser, ensuring it persists across sessions.

Features

Add, edit, and delete income/expense entries.
Filter entries by type (All, Income, Expense).
Calculate and display total income, total expenses, and net balance.
Data persists using local storage.

Technologies Used

HTML: Structure of the app.
CSS (TailwindCSS): Styling for a responsive design.
JavaScript: Logic for CRUD operations and data persistence.

How to Use

Clone or download the project files.
Open the index.html file in your browser.
Add entries, filter by type, and manage your finances.
